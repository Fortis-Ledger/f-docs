# Player Verification (KYC)

To participate in tournaments and earn real rewards, you need to become a **Verified Player** by completing our Know Your Customer (KYC) process.

## Why Verification is Required

### Tournament Integrity

KYC ensures:
- Fair competition (one person = one account)
- Age-appropriate participation
- Legal compliance for prize distribution
- Anti-fraud protection

### What's Required

**Basic Verification:**
- Full legal name
- Date of birth
- Country of residence
- Government ID upload
- Selfie for identity matching

**Optional for Higher Limits:**
- Proof of address
- Additional ID verification
- Phone verification

## Verification Process

<Steps>
  <Step title="Access Verification">
    Go to Settings → Account → Verification
    Or visit Tournament Hub and click "Get Verified"
  </Step>
  
  <Step title="Submit Information">
    Enter personal details exactly as on ID
    Upload clear photos of government ID
    Take real-time selfie for matching
  </Step>
  
  <Step title="Wait for Review">
    Automated review: Instant to 24 hours
    Manual review (if needed): 2-3 business days
  </Step>
  
  <Step title="Verification Complete">
    Receive email confirmation
    Green checkmark appears on profile
    Access Tournament Hub unlocked
  </Step>
</Steps>

### Document Requirements

- ✅ Color photo (not scanned)
- ✅ All four corners visible
- ✅ Not expired
- ✅ Clear, not blurry
- ✅ No glare or shadows
- ✅ Original document (not copy)

## Verification Levels

### Level 1: Basic (Required for Tournaments)

- Identity verification
- Age confirmation (18+)
- Country verification

**Unlocked:**
- Tournament participation
- Squad creation/joining
- Basic wallet features
- Earnings up to $1,000/month

### Level 2: Standard (Higher Limits)

- Address verification
- Phone verification
- Enhanced identity check

**Unlocked:**
- Earnings up to $10,000/month
- Organization applications
- Premium features
- Priority support

### Level 3: Organization (For Orgs)

- Business verification
- Tax documentation
- Legal entity confirmation

**Unlocked:**
- Host tournaments
- Create official guilds
- Unlimited earnings
- API access

## Privacy & Security

### Data Protection

- Encrypted storage (AES-256)
- Access limited to verification team
- Not shared with third parties
- Deleted if account closed
- GDPR compliant

### Security Measures

- Documents stored offline after verification
- Automated identity matching
- Liveness detection (prevent photo spoofing)
- Regular security audits

## Troubleshooting

### Verification Failed

| Reason | Solution |
|:-------|:---------|
| Blurry photo | Retake with better lighting |
| ID expired | Use current valid ID |
| Name mismatch | Use exact name on ID |
| Age under 18 | Wait until eligible |
| Restricted country | Check supported regions |

### Pending Too Long

- Check email for additional requests
- Ensure all documents uploaded
- Contact support if >3 days

## FAQ

**Is my data safe?**
Yes, encrypted and access-controlled. Never sold to third parties.

**Can I change my verification info?**
Contact support for name/country changes. Requires re-verification.

**What if I don't have ID?**
Passport is universally accepted. Apply for government ID if needed.

**Can I play without verification?**
Yes, but only social features. Tournaments require verification.

**How long is verification valid?**
Indefinite, but may require re-verification if suspicious activity detected.
