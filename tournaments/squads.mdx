# Squads & Teams

A **Squad** is a competitive team of 5-7 players (depending on game) that competes together in tournaments. Squads are the foundation of team-based esports on FortisArena.

## What is a Squad?

```
Squad = Team of Verified Players
      ├── Leader (Manager)
      ├── Players (5-7 members)
      ├── Roles (IGL, Support, Entry, etc.)
      └── Stats (Win rate, rankings)
```

## Squad Features

### Core Features

| Feature | Description |
|:--------|:------------|
| **Roster Management** | Add/remove players, set roles |
| **Match History** | Record of all squad matches |
| **Leaderboard** | Rank vs other squads |
| **Recruitment** | Open/closed to new applicants |
| **Trial System** | Test new players before full join |
| **Private/ Public** | Visibility settings |

### Squad Stats Tracked

- Total matches played
- Win/loss ratio
- Tournament placements
- Average K/D (per game)
- Role performance metrics
- Team chemistry score

## Creating a Squad

### Requirements

- ✅ Verified Player status (KYC complete)
- ✅ Minimum Career Level: Amateur
- ✅ At least 1 other member to start

### Step-by-Step

<Steps>
  <Step title="Access Squads">
    Go to Tournament Hub → Squads → Create Squad
  </Step>
  
  <Step title="Basic Info">
    - Squad name (unique)
    - Game(s) you play
    - Region/Server
    - Squad tag (3-4 letters)
  </Step>
  
  <Step title="Settings">
    - Public or Private
    - Open or Application-only recruitment
    - Role requirements
  </Step>
  
  <Step title="Invite Members">
    - Search for verified players
    - Send invitations
    - Minimum 2 members to activate
  </Step>
  
  <Step title="Squad Live">
    Start competing in tournaments!
  </Step>
</Steps>

### Squad Tiers

| Tier | Cost | Features |

## Joining a Squad

### Find a Squad

1. **Browse Public Squads:**
   - Filter by game, region, skill level
   - View squad stats and requirements
   - See open roles

2. **Apply to Join:**
   - Submit application with message
   - Include your stats and availability
   - Wait for leader response

3. **Trial Period:**
   - Some squads require trial matches
   - Prove your skills and teamwork
   - Mutual fit assessment

### Squad Requirements

Common requirements squads set:

| Requirement | Typical Value |
|:------------|:--------------|
| **Career Level** | Amateur+ / Semi-Pro+ |
| **Win Rate** | 45%+ / 50%+ / 55%+ |
| **Availability** | Evenings / Weekends / Flexible |
| **Region** | Same timezone / Server |
| **Role** | Specific position needed |
| **Language** | English / Regional |

## Squad Management

### Roles & Permissions

| Role | Permissions |
|:-----|:------------|
| **Leader** | Full control, disband squad |
| **Co-Leader** | Manage roster, settings |
| **Manager** | Recruitment, scheduling |
| **Player** | Compete, view stats |
| **Trial** | Limited access, being evaluated |

### Best Practices

**For Leaders:**
- Set clear expectations
- Schedule regular practice
- Communicate availability
- Be fair in role assignments
- Handle conflicts professionally

**For Members:**
- Be punctual for matches
- Communicate availability changes
- Practice your role
- Provide constructive feedback
- Support teammates

## Competitive Play

### Entering Tournaments

1. Leader browses tournaments
2. Checks squad eligibility
3. Registers squad
4. All members confirm availability
5. Compete together

### Match Day

- Squad leader receives match details
- Coordinates team arrival
- Submits lineup
- Communicates strategy
- Reports results

### Scrimmages (Scrims)

Practice matches against other squads:
- Schedule via platform
- Custom game lobbies
- Stats tracked
- No prizes (practice only)

## Squad Rankings

### How Rankings Work

Squads are ranked based on:
- Tournament placements
- Win rate
- Match history
- Activity level

### Leaderboards

| Leaderboard | Based On |
|:------------|:---------|
| **Global** | All squads worldwide |
| **Regional** | Same geographic region |
| **Game-Specific** | Per game title |
| **Monthly** | Recent performance |
| **All-Time** | Career achievements |

## Leaving a Squad

### Voluntary Leave

- Give notice to leader
- Finish current tournaments
- Squad stats remain
- Can rejoin if invited

### Removal

Leaders can remove members for:
- Inactivity
- Toxic behavior
- Missing matches
- Violating squad rules

### Squad Disbanding

If leader disbands:
- All members notified
- Ongoing tournaments affected
- Stats archived
- Name becomes available

## FAQ

**Can I be in multiple squads?**
Yes, but only one per game title.

**Can I change squad name?**
Yes, but limited to once per month.

**What if my squad is inactive?**
Inactive squads (30+ days) may be archived.

**Can squads merge?**
Yes, contact support for merger assistance.

**How many players needed?**
Minimum 2 to create, 5-7 for most tournaments.
