## FortisArena Internal Wallet Security

### How Our Wallet is Different

FortisArena uses **MPC (Multi-Party Computation)** technology:

| Feature | Traditional Wallet | FortisArena Wallet |
|:--------|:-------------------|:-------------------|
| **Recovery** | Seed phrase (12-24 words) | Email/Phone verification |
| **Key Management** | You manage private keys | Platform secures keys |
| **If You Forget** | Funds lost forever | We can help recover |
| **Support** | None | Full customer support |

### Your Responsibilities

**You ARE responsible for:**
- Account password security
- 2FA setup and backup
- Email account security
- Verifying withdrawal addresses
- Reporting suspicious activity

**You are NOT responsible for:**
- Private key management
- Seed phrase backup
- Wallet software updates
- Blockchain node operation

If you lose access:

## Recognizing Scams

### Common Scams

<AccordionGroup>
  <Accordion icon="fish" title="Phishing (Fake Websites)">
    **How it works:**
    - Fake website looks identical to real one
    - URL is slightly different
    - Steals credentials when you log in
    
    **Examples:**
    - ‚ùå `fortis-arena.io` (fake)
    - ‚ùå `fortisarena.xyz` (fake)
    - ‚úÖ `fortisarena.io` (real)
    - ‚úÖ `app.fortisarena.io` (real)
    
    **Prevention:**
    - Bookmark official sites
    - Check URL carefully
    - Look for SSL lock (üîí)
  </Accordion>
  
  <Accordion icon="message-circle" title="Fake Support">
    **How it works:**
    - "Support agent" DMs you first
    - Asks for account info or wallet details
    - Offers to "help" with a problem
    
    **Red Flags:**
    - Anyone DMing you first claiming to be support
    - Asking for your password
    - Asking you to connect to "verification" websites
    - Pressure to act quickly
    
    **Remember:** Real FortisArena support will NEVER:
    - DM you first
    - Ask for your password
    - Ask for your seed phrase
    - Ask you to send crypto for "verification"
  </Accordion>
  
  <Accordion icon="gift" title="Giveaway Scams">
    **How it works:**
    - "Send 1 FRT, get 2 FRT back"
    - Fake celebrity endorsements
    - Time pressure to "act now"
    
    **Reality:**
    - If it sounds too good to be true, it is
    - No one gives away free crypto for sending crypto
    - Legitimate giveaways never require sending funds first
  </Accordion>
  
  <Accordion icon="code" title="Fake Apps">
    **How it works:**
    - Malicious browser extension
    - Fake mobile apps
    - Modified wallet software
    
    **Prevention:**
    - Only download from official sources
    - Check reviews and download counts
    - Verify publisher identity
  </Accordion>
</AccordionGroup>

### Red Flags - STOP Immediately If You See:

<Warning>
  **Never proceed if:**
  - Someone asks for your password
  - Someone asks for your seed phrase (FortisArena doesn't use these)
  - Promises of guaranteed returns
  - Unsolicited DMs offering help
  - Requests to install remote access software
  - "Verify" by sending crypto
</Warning>
