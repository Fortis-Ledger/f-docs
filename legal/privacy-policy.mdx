## 3. FortisArena Wallet System

### 3.1 UID System (Internal Identifiers)

FortisArena uses a **UID (User Identifier)** system instead of exposing blockchain addresses for internal transactions:

- **Your UID**: `FORTIS-UID-XXXX` format
- **Used for**: Internal transfers, tips, rewards
- **Visibility**: Public (shown on profile)

### 3.2 Wallet Security (MPC Technology)

Our internal wallet uses **Multi-Party Computation (MPC)** technology:

| Feature | How It Works |
|:--------|:-------------|
| **No Seed Phrases** | Account recovery via email/phone verification |
| **Key Distribution** | Private keys split across secure nodes |
| **Platform Custody** | We secure the keys; you don't manage them |
| **Recovery** | Email/SMS-based account recovery |

<Warning>
  **Important**: Unlike traditional crypto wallets, FortisArena does NOT provide seed phrases. Account recovery is done through standard email/phone verification.
</Warning>

### 3.3 Blockchain Data

For on-chain transactions (deposits/withdrawals):

- **Public Blockchain Data**: Wallet addresses and transaction hashes are recorded on BNB Smart Chain (public)
- **Internal Ledger**: UID-to-UID transfers are recorded on our secure internal ledger
- **User Control**: You can withdraw to external wallets anytime
