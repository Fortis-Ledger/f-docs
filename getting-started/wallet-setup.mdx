
**Use UID for:**

## Common Questions

### Do I need MetaMask?

**No!** MetaMask is completely optional:

- âœ… Internal wallet works for everything
- âœ… Deposits/withdrawals work without MetaMask
- âœ… Staking works with internal wallet
- âœ… Only use MetaMask if you need external DeFi access

### What if FortisArena shuts down?

**Your on-chain funds are safe:**
- FRT/FPT are on blockchain (BSC)
- Your ownership is recorded on-chain
- Can withdraw to external wallet anytime
- Fiat balances are held in regulated accounts

### Can I see my private key?

**No!** And this is a **security feature**:

- Private keys are distributed across secure nodes
- Users don't need to manage them
- Hacking risk is significantly reduced
- Professional security management

### Is there a mobile wallet app?

**YES!** FortisArena mobile app includes full wallet:
- iOS app
- Android app
- Same features as web
- Biometric security

### Can I have multiple wallets?

**One UID per account**, but you can:
- Connect multiple external wallets
- Use same UID across devices
- Transfer between your own wallets

## Troubleshooting

### Deposit Not Showing

| Check | Solution |
|:------|:---------|
| **Confirmations** | Wait for 15+ BSC confirmations |
| **Network** | Ensure sent on BSC (not Ethereum) |
| **Address** | Double-check deposit address |
| **Time** | Up to 30 minutes is normal |

**Still missing after 1 hour?**
- Contact support with transaction hash
- Email: wallet@fortisarena.io

### Withdrawal Stuck

| Status | Meaning | Action |
|:-------|:--------|:-------|
| **Pending** | Awaiting processing | Wait (up to 24h for fiat) |
| **Processing** | Being sent | Wait for confirmations |
| **Completed** | Sent successfully | Check destination wallet |
| **Failed** | Transaction failed | Contact support |

### Can't Access Wallet
