## ğŸš« Common Issues

### Didn't Receive Verification Email

1. Check spam/junk folder
2. Wait 5 minutes
3. Click "Resend" on signup page
4. Try different email provider
5. Contact support if still missing

### "Email Already Exists"

- Try logging in instead
- Use "Forgot Password" if needed
- Check if you used Google OAuth previously

### Can't Complete KYC

| Issue | Solution |
|:------|:---------|
| ID rejected | Ensure clear photo, good lighting |
| Selfie doesn't match | Remove glasses, ensure good lighting |
| Country not supported | Check [supported countries list](/tournaments/verification) |
| Under 18 | Must be 18+ for KYC verification |
