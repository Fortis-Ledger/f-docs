## üí∞ Wallet System

### Does the platform provide its own wallet?

**YES!** FortisArena has its own **internal wallet**.

<Info>
  **NO EXTERNAL WALLET NEEDED**
  
  ‚úÖ Platform wallet auto-creates on signup  
  ‚úÖ No MetaMask  
  ‚úÖ No Trust Wallet  
  ‚úÖ No seed phrases  
</Info>

### Does the user get a seed phrase at signup?

**NO!** No seed phrase is given. The platform uses **MPC (Multi-Party Computation)** technology:

| Traditional Wallet | Fortis Wallet |
|:-------------------|:--------------|
| Seed phrase (12-24 words) | ‚ùå Not available |
| User responsible for backup | ‚úÖ Platform handles |
| Lose phrase = Lose funds | ‚úÖ Account recoverable |
| External app needed | ‚úÖ Built-in platform |

**Recovery:** Account is recovered through Email/Phone verification.

### Can the user see their private key?

**NO!** Private keys are stored on the platform's secure infrastructure:

- Keys distributed across multiple secure nodes
- User doesn't need to manage them
- Security handled by platform
- Account recovery through standard verification

### Can external wallets (MetaMask) also be connected?

**YES, but optional:**

| Feature | Internal Wallet | MetaMask (Optional) |
|:--------|:----------------|:--------------------|
| **Signup** | Auto-create | Connect later |
| **Daily use** | ‚úÖ Recommended | Not needed |
| **Withdraw crypto** | ‚úÖ Works | ‚úÖ Works |
